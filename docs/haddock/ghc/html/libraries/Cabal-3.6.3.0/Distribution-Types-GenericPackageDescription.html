  • gpdScannedVersion :: Maybe Version

    This is a version as specified in source. We populate this field in index reading for dummy GPDs, only when GPD reading failed, but scanning haven't.

    Cabal-the-library never produces GPDs with Just as gpdScannedVersion.

    Perfectly, PackageIndex should have sum type, so we don't need to have dummy GPDs.

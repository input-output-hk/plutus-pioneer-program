beta :: Term tyname name uni fun a -> Term tyname name uni fun a Source #

Recursively apply the beta transformation on the code, both for the terms

    ( (x : A). M) N
    ==>
    let x : A = N in M

and types

    (/ a. (x : a) . x) {A}
    ==>
    let a : * = A in
    ( (x : A). x)
